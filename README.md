# PreventMultipleTabs

JavaScript that attempts to **enforce a single active session by blocking multiple open tabs.**

---

## Features

- **Easy to setup** 

---

## Use Cases

- Websites/Apps that intend to prevent users from flooding/gaming the system i.e ticketing platforms.
